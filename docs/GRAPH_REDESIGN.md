# グラフ表示方式の見直し（2026-05-26）

Windows 版の考え方は残しつつ、Web 版のグラフ表示だけを作り直した。

## 変更前

- ピッチ推定結果を配列に入れ、点数を横幅いっぱいに均等配置
- 声が取れないときも `0 oct` として追加
- 横軸は厳密な時間ではなく、直近の点数

## 変更後

- 各点に `timeMs` を持たせ、直近 12 秒を実時間で表示
- 声が取れない点は線を切る
- 基準 Hz、目標オクターブ、縦軸オクターブ表示は従来どおり
- 目盛りとして `-2s`, `-4s` などの時間線を追加
